This page uses a **dark mode** color scheme, primarily featuring a very dark gray background and light text, with a **vibrant purple** as the accent color.

About the Colors

The color palette is inspired by material design's dark theme, where the main accent color (`var(--accent-color)`) is a bright, easily visible purple (`#BB86FC`) that stands out against the dark background. This ensures **accessibility** and visual appeal.

Styling Details

  • **Background:** Dark gray (`#121212`) for a true dark mode.
  • **Text:** Light gray (`#E0E0E0`) for contrast.
  • **Accent:** A bright, bold purple (`#BB86FC`).
  • **Containers:** A slightly lighter dark gray (`#1E1E1E`) to separate content blocks.
